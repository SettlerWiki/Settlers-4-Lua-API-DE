# Game.GetMapNameRelativePath

## `Game.`GetMapNameRelativePath`()`

Gibt den relativen Kartennamen zurück.

#### Rückgabewert

Relativer Kartenname

#### Beispiel

```lua
local mapNameRelativePath = Game.GetMapNameRelativePath()
```
