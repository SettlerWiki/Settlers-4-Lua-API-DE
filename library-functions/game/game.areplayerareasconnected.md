# Game.AddBeds

## `Game.AddBeds(playerID, amount)`

Fügt dem angegebenen Spieler \[playerID] eine Anzahl \[amount] an Betten hinzu, die auch ohne Wohnhäuser bestehen bleiben.

#### Rückgabewert

None

#### Beispiel

```lua
Game.AddBeds(1, 100)
```
