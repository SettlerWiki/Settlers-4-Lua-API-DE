# SU.Game.GetMapNameRelativePath

## SU.Game.GetMapNameRelativePath()

Gibt den relativen Kartennamen zurück.

#### Rückgabewert

* Relativer Kartenname

#### Beispiel

```lua
local mapNameRelativePath = SU.Game.GetMapNameRelativePath()
```
