# SU.Map.GetObject

## SU.Map.GetObject(x, y)

Gibt das Objekt an den angegebenen Koordinaten zurück.

#### Parameter

* `x, y`: Koordinaten

#### Rückgabewert

* [su.objects.md](../../su-api-enums/su.objects.md "mention")

#### Beispiel

```lua
local objID = SU.Map.GetObject(x, y)
```
