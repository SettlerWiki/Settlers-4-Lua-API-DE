# SU.Map.GetResource

## SU.Map.GetResource(x, y)

#### Rückgabewert

* [ai-7.md](../../su-api-enums/ai-7.md "mention")

#### Beispiel

```lua
local resource = SU.Map.GetResource(x, y)
```
